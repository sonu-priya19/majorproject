How it works

This tool uses a Random Forest model trained on URL features (length, dots, https, keywords, IP, TLD, etc.). It does not visit the page.

  • Paste a URL on the Scan page.
  • We extract features and run the model.
  • We show SAFE/PHISHING and a confidence score.

💡 Use the chatbot (bottom-right) to ask questions about results and safety tips.
